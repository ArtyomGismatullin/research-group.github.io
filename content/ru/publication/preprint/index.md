abstract: На первом этапе проекта исследуется модель колебаний одномерных цепочек частиц в рамках математического моделирования. Теоретически описаны гармоническая цепочка (линейная модель с независимыми модами, стоячими волнами и дисперсионными соотношениями) и ангармоническая цепочка (нелинейная модель задачи Ферми-Паста-Улама). В последней введён нелинейный член, приводящий к отсутствию термализации энергии и квазипериодическому поведению системы. Основная научная проблема — выявление условий перехода к тепловому равновесию, включая влияние числа частиц, величины нелинейности и начальных условий. Для анализа использовано преобразование Фурье, позволяющее изучать спектральное распределение энергии. Результаты этапа подчёркивают фундаментальное различие между линейными и нелинейными системами, а также ставят вопросы для дальнейшего исследования динамики сложных колебательных систем.

# 2 Гармоническая цепочка: модель и решение

Гармоническая цепочка — одномерная модель твердого тела, состоящая из **N частиц** массой **m**, соединенных пружинами жесткости **k**. Крайние частицы закреплены на стенках, общее число пружин: **N + 1** (длина каждой — **d**).  
Равновесные положения частиц:  
`x i = i·d`  
Смещения от равновесия: **y i** (предполагается **y i << d**).

---

### Сила и уравнение движения
- **Сила**, действующая на **i-ю частицу**:  
  `F i = k·(y i+1 - y i) - k·(y i - y i-1) = k·(y i+1 - 2·y i + y i-1)`

- **Уравнение движения**:  
  `m·d²y i/dt² = k·(y i+1 - 2·y i + y i-1)`  
  (для `i = 1, 2, ..., N`)

---

### Энергия системы
Полная энергия включает:  
```
U = (m/2)·Σ(dy i/dt)² + (k/2)·Σ(y i - y i-1)²
```
- Σ — сумма от `i = 1` до `N` (кинетическая энергия) и до `N + 1` (потенциальная).

---

### Решение: стоячие волны
Смещения частиц:  
`y i = [A·cos(p·x i) + B·sin(p·x i)]·cos(ω·t)`  
где:  
- `p` — волновое число,  
- `ω` — частота.

**Граничные условия** (`y 0 = 0`, `y N+1 = 0`):  
- `A = 0`,  
- `sin(p·(N + 1)·d) = 0` → допустимые волновые числа:  
  `p l = l·π / [(N + 1)·d]` (`l = 1, 2, ..., N`).

**Дисперсионное соотношение**:  
`ω l = 2·ω 0·sin(l·π / [2·(N + 1)])`  
(здесь `ω 0 = sqrt(k/m)` — базовая частота).

---

# 3 Ангармоническая цепочка: задача Ферми-Паста-Улама

### Нелинейная сила
Сила включает ангармонический член:  
`F = -k·x·(1 - α·x/d)`  
где **α** — коэффициент нелинейности.

**Потенциальная энергия**:  
```
U = (k/2)·Σ(y i - y i-1)² - (k·α)/(3·d)·Σ(y i - y i-1)³
```

---

### Результаты задачи FPU
- **Ожидалось**: энергия распределится по всем модам.  
- **Реальность**:  
  - Энергия циркулирует между несколькими модами (для `N = 32` моды `l > 5` получали < 1% энергии).  
  - **Квазипериодичность**: система возвращалась к начальному состоянию через ~160 периодов.  
  - **Сверхпериодичность**: восстановление 98% энергии через большие интервалы времени.

---

# 4 Преобразование Фурье для анализа

### Непрерывное преобразование
Периодическая функция `f(x)` с периодом **L**:  
```
f(x) = a 0/2 + Σ[a l·cos(2π·l·x/L) + b l·sin(2π·l·x/L)]
```
(сумма от `l = 1` до ∞).

---

### Дискретное преобразование (DFT)
Для `N` точек `x j = j·L/N`:  
```
f N(x) = a 0/2 + Σ[a l·cos(2π·l·x/L) + b l·sin(2π·l·x/L)] + a n/2·cos(2π·n·x/L)
```
(сумма от `l = 1` до `n - 1`).

---

# 5 Научная проблема

**Ключевой вопрос**:  
_Какие условия обеспечивают тепловое равновесие в колебательных системах?_  

- **Гармоническая цепочка**: моды независимы → энергия не перераспределяется.  
- **Ангармоническая цепочка**: нелинейность ≠ гарантия термализации (пример FPU).  

**Факторы влияния**:  
1. Число частиц (**N**),  
2. Уровень нелинейности (**α**),  
3. Начальные условия.
